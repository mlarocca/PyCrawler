Welcome to PyCrawler's documentation!
=====================================

PyCrawler is a Breadth-first multithread crawler

This projects depends on the following packages:

    * Queue
    * urllib2
    * urlparse
    * time
    * Set
    * threading
    * datetime
    * logging
    * HTMLParser import HTMLParser
